## API

属性 | 说明 | 类型 | 默认值 | 必选
----|-----|------|------|------
initialKey | 初始化选中的key | string | '' | false
data | 数据源 | any[] | [] | false
selectedColor | 选中项的颜色 | string | '' | false
extraTextColor | 额外文本的颜色 | string | '' | false
modelMarginTop | 弹窗顶部外边距 | number | 0 | false
onSelectItem | 选中某项回调 | (item) => void | 无 | false

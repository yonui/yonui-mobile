## API

属性 | 说明 | 类型 | 默认值 | 必选
----|-----|------|------|------
content | 提示内容 | React.Element/String | 无 | false
duration | 自动关闭的延时,单位秒 | number | 3 | false
onClose | 关闭后回调 | () => void | 无 | false
mask | 是否显示透明蒙层,防止触摸穿透 | boolean | true | false

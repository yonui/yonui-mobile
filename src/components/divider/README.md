## API

属性 | 说明 | 类型 | 默认值 | 必选
----|-----|------|------|------
content | 内容 | string | '' | false
prefixCls | 样式前缀 | string | 'mdf-divider' | false
width | 线宽 | number | 1 | false
color | 颜色 | string | '#d0d0d0' | false
visible | 是否可见 | boolean | true | false
style | 样式style | React.CSSProperties | 无 | false

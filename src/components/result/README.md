## API

属性 | 说明 | 类型 | 默认值 | 必选
----|-----|------|------|------
mode | 模式 success: 成功 wait: 等待 reject: 驳回 | string | 'success' | false
message | 文字描述 | string | '成功' | false
style | 样式 | React.CSSProperties | 无 | false

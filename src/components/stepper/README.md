## API

属性 | 说明 | 类型 | 默认值 | 必选
----|-----|------|------|------
stepType | 元数据字段 | select | 1 | false
max | 最大值 | number | 100 | false
min | 最小值 | number | 0 | false
defaultValue | 默认值 | number | 0 | false
step | 步长 | number | 1 | false
disabled | 是否禁用 | boolean | false | false
onChange | 值改变时回调 | () => void | 无 | false
